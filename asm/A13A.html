A13A: Morale flags.
Stops set_hero_target_location working.
Used to set flag colour.
morale_1 and morale_2 are treated as a word by process_player_input. Everything else treats them as bytes.
Read-only by process_player_input, in_permitted_area, set_hero_target_location, follow_suspicious_character.
Write/read-write by charevnt_handler_4_zeroes_morale_1, solitary.
Input
Output
A13A DEFB $00 morale_1
Inhibits user input when non-zero.
Set by check_morale.
Reset by reset_game.
Read-only by process_player_input.
Write/read-write by check_morale.
A13B DEFB $00 morale_2
