CCFB: is_item_discoverable_interior
Examines the specified room to see if it contains a discoverable item.
A discoverable item is one moved away from its default room, and one that isn't the red cross parcel.
A Room ref.
Flags Z => found, NZ => not found.
C Item (if found).
CCFB LD C,A C = A; // room ref
CCFC LD HL,$76C9 HL = &item_structs[0].room; // pointer to room ref
CCFF LD B,$10 B = 16; // items__LIMIT
CD01 LD A,(HL) do { A = *HL & itemstruct_ROOM_MASK; } { if (A == C) { // yes } - { A = default_item_locations[HL[-1] & itemstruct_ITEM_MASK].room_and_flags; // HL[-1] = itemstruct.item } { if (A != C) goto not_in_default_room; } - } { next: HL += 7; // stride } while (--B);
CD02 AND $3F
Is the item in the specified room?
CD04 CP C
CD05 JR NZ,$CD1B
CD07 PUSH HL
Has the item been moved to a different room?
Note that room_and_flags doesn't get its flags masked off.
CD08 DEC HL
CD09 LD A,(HL)
CD0A AND $0F
CD0C LD E,A
CD0D ADD A,A
CD0E ADD A,E
CD0F LD E,A
CD10 LD D,$00
CD12 LD HL,$CD6A
CD15 ADD HL,DE

CD17 CP C
CD18 JR NZ,$CD22
CD1A POP HL
CD1B LD DE,$0007
CD1E ADD HL,DE
CD1F DJNZ $CD01
CD21 RET return; // return with NZ set
CD22 POP HL not_in_default_room: -
CD23 DEC HL A = HL[-1] & itemstruct_ITEM_MASK; // itemstruct.item
CD24 LD A,(HL)
CD25 AND $0F
Ignore red cross parcel.
